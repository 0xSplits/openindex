# OpenIndex - An npm package template.

## Overview

OpenIndex is a template for creating developer-friendly Typescript packages. It's forked from a library called [Ox](https://github.com/wevm/ox).

This page contains instructions on getting started with OpenIndex, including setting up a documentation site like this one (it's all generated from JSDocs in your code). The rest of this site is example pages to show you what's possible. Feel free to use these pages as a starting point, or start from scratch. The documentation framework used is [Vocs](https://github.com/wevm/vocs), which you can learn more about from [their documentation](https://vocs.dev).

:::info
Openindex was forked from a library called [Ox](https://github.com/wevm/ox). All credit for the project structure goes to the Ox team.
:::
